# PWA_Projekt
Kristina Bobek PWA Projekt 
L'OBS portal za vijesti

XAMPP, CSS, HTML, PHP, MySQL

1. Importati lobs.sql bazu podataka u localhost/phpMyAdmin
2. Kopirati projekt u xampp/htdocs
3. Pokrenuti Apache i MySQL server
4. Otvoriti localhost/projektKB/index.php

!! VAŽNO !!
Na osobnom računalu koristim port 3307 za MySQL.
U datoteci 'connect.php' treba obrisati taj broj ako na vlastitom računalu ne koristite port 3307, inače neće raditi.
!!
